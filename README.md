# Tic-Tac-Toe

A simple Tic-Tac-Toe game that allows 2 players to play against each other and keeps track of their scores.

## Quick Start

1. **Install dependencies:**

   ```bash
   make init
   ```

2. **Run the game locally:**

   ```bash
   make start
   ```

3. **Build the game:**

   ```bash
   make build
   ```
